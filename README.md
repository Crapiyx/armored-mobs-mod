# Armored Mobs — Fabric Mod pour Minecraft 26.1.2

## IMPORTANT : Prérequis

- **Java 25** (obligatoire pour 26.1.2) → https://adoptium.net/temurin/releases/?version=25
- **Fabric Loader 0.18.4** → https://fabricmc.net/use/installer/
- **Fabric API** → https://modrinth.com/mod/fabric-api (prends la version pour 26.1.2)

⚠️ Java 21 ne suffira PAS — Minecraft 26.1.2 requiert Java 25 minimum.

## Compilation

1. Installe Java 25
2. Place le gradle-wrapper.jar dans gradle/wrapper/ (voir ci-dessous)
3. Lance : `.\gradlew.bat build`
4. Le .jar sera dans `build/libs/armored-mobs-1.0.0.jar`

## Installation

Copie `armored-mobs-1.0.0.jar` dans ton dossier `mods/` avec Fabric Loader et Fabric API.
